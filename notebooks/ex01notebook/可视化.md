
|---|---|---|
| `waveform`（波形图） | 显示音频振幅随时间的变化 | 检查音频是否有声音、是否存在静音或异常振幅 |
| `spectrogram / CQT`（频谱图 / 常数 Q 变换图） | 显示音频在不同时间和频率上的能量分布 | 检查频谱结构是否正常，是否具有音乐信号特征 |
| `piano-roll`（钢琴卷帘图 / 音符时序图） | 显示 MIDI 音符在时间和音高上的分布 | 检查 MIDI 标注是否包含合理的音符事件 |
| `audio duration vs MIDI duration`（音频时长与 MIDI 时长对比） | 比较音频文件和 MIDI 文件的总时长 | 判断 audio-MIDI 是否大致对齐，结尾差异是否来自尾音 |
| `piano vs organ comparison`（钢琴与管风琴对比） | 对比钢琴和管风琴样本的音频表示 | 观察音色变化对 AMT 输入特征的影响 |

## Piano 样本观察结果

抽查的 MAESTRO piano 样本显示，audio 与 MIDI 在开头和结尾均基本对应。

| 观察内容 | 结果 |
|---|---|
| 开头 waveform | 音频有正常振幅变化，不是静音或损坏文件 |
| 开头 CQT | 存在清晰的音乐频谱结构 |
| 开头 piano-roll | MIDI 音符分布正常，音高范围合理 |
| 结尾 waveform | MIDI end 附近音频自然衰减 |
| 结尾 CQT | MIDI end 后没有明显有效频谱能量 |
| 结尾 piano-roll | MIDI 音符在 MIDI end 前正常结束 |
| audio-MIDI 时长差 | 抽查样本均小于 1 秒 |

## Organ 样本观察结果

抽查的 organ 样本显示，audio 与 MIDI 在主体部分基本对应，但 audio 文件结尾相对于 MIDI 文件存在约 2.4–2.5 秒的固定尾部。

示例 organ 样本：

| 样本 | audio 时长 | MIDI 时长 | 差值 | note 数 |
|---|---:|---:|---:|---:|
| `faf_gm11` | 350.064s | 347.627s | 2.437s | 1387 |

该样本的可视化结果如下：

| 检查内容 | 观察结果 |
|---|---|
| 开头 waveform | 音频有正常振幅变化，主体部分不是静音 |
| 开头 CQT | 存在清晰的音乐频谱结构 |
| 开头 piano-roll | MIDI 音符分布正常，并与 CQT 的时间结构大致对应 |
| 结尾 waveform | MIDI end 之后音频基本接近静音 |
| 结尾 CQT | MIDI end 之后没有明显稳定频谱能量 |
| 结尾 piano-roll | MIDI 在 MIDI end 前正常结束 |

结论：organ 的 `duration_diff` 主要来自音频末尾的固定尾部或近静音 padding，而不是文件配对错误。

## Piano 与 Organ 的差异

从可视化结果看，piano 与 organ 在音频表现上存在明显差异。

| 对比项 | Piano | Organ |
|---|---|---|
| waveform | 击键后有明显动态变化和自然衰减 | 声音持续性更强，波形更平稳 |
| CQT | 谐波和瞬态变化较明显 | 横向持续亮带更明显 |
| 结尾差异 | audio-MIDI 时长差通常小于 1 秒 | audio 比 MIDI 稳定多出约 2.4–2.5 秒 |
| 可能原因 | 原始录音和官方对齐较完整 | MIDI 合成音频时保留了固定尾部或静音 padding |

该差异说明，organ 数据的尾部差异更可能来自合成流程，而不是数据本身严重错误。

## 关于 organ 固定尾部的处理

内容检查中最初使用：

`duration_diff_sec <= 1.0`

作为 audio-MIDI 时长匹配标准。该标准将 15 条 organ 样本标记为 `duration_mismatch`。

进一步检查发现：

| 观察 | 说明 |
|---|---|
| organ 的时长差均集中在约 2.4–2.5 秒 | 差值稳定 |
| 所有异常均为 audio 比 MIDI 长 | 方向一致 |
| 可视化显示 MIDI end 后基本为静音或近静音 | 不是有效演奏内容 |
| 开头 audio、CQT 和 piano-roll 基本对应 | 不像文件配错 |

因此，本实验将时长匹配阈值放宽为：

`duration_diff_sec <= 3.0`
